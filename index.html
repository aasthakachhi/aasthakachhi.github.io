Research Scholar

Ministry of Electronics and Information Technology| June 2022 – Present
  • To develop and implement Assistive Speech Technologies for the differently abled class, using signal processing, Acoustical processing and Machine Learning algorithms.
  • Conceptualization and implementation of Neural network/machine learning/Deep Learning algorithms for pathological infant cry detection to reduce the infant mortality rate. 97% Accuracy was achieved for the same.
  • Transfer Learning based whisper model was implemented for multi-class pathology detection. Accuracy was increased by 2% to give a 99% classification rate.
  • Acoustical Analysis of normal and pathological cries was conducted for 100+ cries

  • Generative Adversarial Networks(GANs) were studied for dysarthric speech enhancement and infant cry data augmentation.
  • Application Programming Interfaces were developed for the practical deployment of implemented infant cry and dysarthric classification systems.
